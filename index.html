Image recolorization

I was curious if we would train a convolutional neural network to predict color from a gray scale image. The results were a bit mixed. While it is possible to do this on a very restrictive domain, such as the MPI sintel, it doesn't work at all on general images (eg COCO or Imagenet). A quick summary of this can be found here.
